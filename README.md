## Overview

**Dena Pawna** is an Android money borrow tracking application
designed to deliver a simple, fast, and intuitive experience for creating and managing notes.
Built with **Android Studio** and **Jetpack Compose**,
the app follows clean and maintainable Android development practices suitable for learning projects.
Dena Pawna focuses on simplicity, usability, and performance.

---

## Features

* Create, edit, and delete people
* Add entries who borrowed what.
* Clean, minimal, and intuitive UI
* Fast performance with local data storage
* Lightweight and responsive design
* Supports Android 10 and above

---

## Tech Stack

* **Language:** Kotlin
* **IDE:** Android Studio
* **Platform:** Android
* **UI Framework:** Jetpack Compose
* **Architecture:** Clean Architecture
* **Local Storage:** SQLite

## Getting Started

### Prerequisites

* Android Studio (latest stable version recommended)
* Android SDK
* Android Emulator or physical Android device
* Minimum Android version: **Android 10 (API 29)**

### Installation

1. Clone the repository:

```bash
git clone https://github.com/AbrarShakhi/dena-pawna.git
```

2. Open the project in **Android Studio**
3. Allow Gradle to sync dependencies
4. Run the app on an emulator or connected device
